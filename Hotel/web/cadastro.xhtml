     Nessa página, temos um formulário com 14 campos, sendo que todos têm um rótulo ao lado informando qual o dado que deve ser preenchido naquele espaço.

     É importante ressaltar que todos os campos devem ser informados corretamente para que não haja dados inválidos salvos no site. Dessa forma, é necessário muita atenção na hora de preencher os dados cadastrais. O campo referente à data de nascimento, é preciso preencher na forma: DD/MM/AAAA, e os campos referentes aos telefones, só são aceitos números.

     Assim, após entrar com os dados válidos, só falta clicar no botão "Cadastrar" para ir à página de confirmação da reserva.
